## 二、司机/车队管理（v3.1重构）✅ 已完成

### 2.1 数据库变更

**新增车队表**：`logistics_fleet`
- 车队ID、车队名称、车队老板姓名、老板联系电话
- 车队开票账户名称、车队开票账号、车队开户行
- 状态、备注等

**修改司机表**：`logistics_driver`
- 删除字段：`driver_code`（司机编码）
- 新增字段：`driver_type`（司机类型）、`fleet_id`（所属车队ID）
- 删除原有车队相关字段（fleet_owner_name等）

### 2.4 核心功能

1. **树形表格展示**
   - 车队（可展开）显示所属车队司机
   - 个人司机直接在根层级显示
   - 图标区分：🏢 车队、👤 个人司机

2. **双新增按钮**
   - 新增车队：打开车队信息表单
   - 新增司机：打开司机信息表单

3. **智能编辑功能**
   - 车队行：点击编辑车队信息
   - 司机行：点击编辑司机信息

4. **搜索功能**
   - 支持搜索车队名称
   - 支持搜索司机姓名

5. **删除保护**
   - 删除车队前检查是否有关联司机
   - 有司机关联的车队无法删除

### 2.5 业务规则

**个人司机**：
- 有个人银行账户（银行账号、开户行、账户姓名）
- 运费直接结算到个人账户

**车队司机**：
- 关联到车队
- 无个人银行账户
- 运费结算到车队账户

**车队**：
- 可以有多个司机
- 有独立的结算账户信息
- 删除前需要检查是否有司机关联
